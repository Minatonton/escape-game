## リスクと対策

| リスク | 影響度 | 対策 |
|--------|--------|------|
| スケジュール遅延 | 高 | Phase 2 で部屋数を減らしてスコープ調整 |
| パズルの難易度バランス失敗 | 中 | 外部プレイテストを Phase 3 の早い段階で実施 |
| アセット品質が間に合わない | 中 | 仮アセット → 本番差し替えの順序を守る |
| Steam 審査リジェクト | 低 | 事前に Steamworks ドキュメントを精読し、禁止コンテンツを含まないよう設計 |
| 一人開発での燃え尽き | 高 | マイルストーンを小さく区切り、達成感を積み重ねる |
